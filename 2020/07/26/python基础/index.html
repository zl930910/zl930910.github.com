python是一种跨平台的计算机程序设计语言。 是一个高层次的结合了解释性、编译性、互动性和面向对象的脚本语言

python和Java又有什么区别呢？

1、他们在声明变量的时候，python是动态类型，而Java是静态类型，就是说java声明了变量后是类型是不可以改变的，而Python则是改变的
2、python一切皆对象，而Java中至少函数，基本数据类型都不算对象。

####python的优点和缺点

优点：python提供了非常完善的基础代码库，覆盖率网络、文件、GUI、文本等大量的内容
除了基本代码库之外还包括有第三方库，提高开发的速度，代码量少
python的缺点：
1、运行速度比较慢
2、代码不能加密

默认参数

设置默认参数需要注意的点
1、必选参数放前面、默认参数放在后面
2、变化大的参数放在前面，变化小的放在后面，设置为默认参数
3、默认参数必须指向不变的对象

可变参数

可变参数指传入的参数的个数可以变，可以是1个参数、2个参数、也可以是0个参数

接收的参数是一个tuple或list类型的参数

关键字参数允许传入0个或多个带参数名称的参数，这个参数的类型是dict字典类型
