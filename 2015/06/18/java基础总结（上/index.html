java概述

简单的说java是一种跨平台的面向对象的程序设计语言。java 技术具有卓越的通用性、高效性、平台移植性和安全性，广泛应用于个人PC、数据中心、游戏控制台、科学超级计算机、移动电话和互联网，同时拥有全球最大的开发者专业社群。

java的特点

  1. 面向对象： 在java中的一切都是对象，容易扩展。
  2. 平台无关性： java运行在JVM虚拟机上，不受平台的约束。
  3. 安全性： java不支持指针，有相应的内存处理机制，安全性高。
  4. 多线程： Java是第一个在语言级提供内至多线程支持的高级语言
  5. 内存管理 Java对内存自动进行管理并进行垃圾回收

java的基础语法

  1. 关键字： 某种语言被赋予了特殊含义的单词，this。
  2. 标识符： 程序中自定义的名词（注：不能以数字开头，不可使用关键字）
  3. 常量： 程序运行中不可变的值。
  4. 变量： 程序运行中可变的值。
  5. 数据类型：基本数据类型：char、byte、int、long 、short、double、float、Boolean 引用数据类型：数组、类、接口
  6. super和this的区别：this调用当前类对象，super调用父类对象。
  7. 作用域 public,protected,private,以及不写时的区别：
    pulic:所有地方都可以访问
    protected：不同包的子类，同一包、当前内可以访问
    private：当前类能访问
    默认：同一包，当前类
  8. final关键字：注：属性不可变，方法不可重写，类不可继承。

面向对象的基本特征

  1. 抽象：找共性，将共有的属性和方法放到父类中。
  2. 继承：子类继承父类，子类拥有父类的属性和方法，可以重用也可以覆盖。
  3. 封装：一个类可以有多个属性和方法。
  4. 多态：可以有多种形态来展示，重载和重写。
