安装准备

  1. 下载node.js
    https://nodejs.org/en/
  2. 下载git
    http://git-scm.com/download/
  3. github或gitcafe账号
    https://github.com/your-user-name
  4. 下载sublime text
    http://www.sublimetext.com/3

创建SSH

创建后，将C:\Users\Administrator.ssh目录 中的id_rsa.pub的内容复制到github中SSH keys中

初始化hexo

1.新建文件夹，Blog,选中文件夹右键运行git bash,运行 git init 自动生成文件

2.在该文件下运行 npm install
3.启动 hexo 本地服务

4.hexo启动后，在浏览器中输入 http://localhost:4000/访问
5.关闭服务

6.新建文章

注：生成的文件放在_posts中，另外，在新建博客前需要停止服务，否则会新建两篇一样的博客
7.生成静态页面

8.上传到github
