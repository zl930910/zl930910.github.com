前言

开年有许多空闲时间，年前就计划学习移动端的自动化测试，当然公司也恰好需要做移动端的自动化测试，正好也是学习的机会。接下来我将分享或是记录我的学习心得。

什么是appium

  • 首先需要明确什么是appium，我理解的appium就是一个从webDriver扩展出来的一个跨平台、跨设备、并支持原生、混合以及web项目的开源的移动端测试框架。

###（一） window7中搭建appium环境

  • 1、安装node.js
  • 安装路径：https://nodejs.org/en/
  • 2、安装appium
  • 安装完node.js后，打开cmd，输入 npm install -g appium 安装appium
  • 3、启动appium

（二）安装安卓环境

  • 1、安装JDK
  • 2、配置jdk环境变量
  • 3、安装sdk并配置sdk环境
  • 4、安装安卓模拟器
    参考博客：http://www.cnblogs.com/fnng/p/4540731.html
    以上是appium以及安卓环境的安装， 在官网上也有详细的介绍，具体请参考appium官方文档http://appium.io/slate/en/v1.6.0/?java#if-you-39-re-running-appium-from-source
