环境准备

在mac上使用appium进行UI自动化前需要准备以下环境：

  1. 安装jdk1.7+ :http://www.cnblogs.com/iOS-mt/p/5726380.html 配置jdk环境变量
  2. xcode 8.3.1：在App Store中下载最新xcode
  3. appium 1.6.3：使用命令安装appium1.6.3
  4. ios版本 10+：准备一台10以上版本的ios真机或者模拟器（我用的是模拟器）

环境安装

下面是安装appium用到的相关依赖以及appium1.6.3版本

  • 安装brew : usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”
  • 安装libimobiledevice：brew install libimobiledevice --HEAD
  • 安装carthage：brew install carthage
  • 安装cnpm：npm install -g cnpm --registry=https://registry.npm.taobao.org
  • 安装ios-deploy：cnpm install -g ios-deploy
  • 安装xcpretty：gem install xcpretty
  • 安装appium1.6.3（注：目前1.6.3是没有图形界面的）：cnpm install -g appium@1.6.3
  • 安装wd：npm install wd
  • 安装npm install -g appium-doctor`
    完成后，使用命令：appium-doctor检查appium环境，我在此之前我安装了安卓的环境，所以一切都ok了。

    以上表示appium的相关环境完成安装，若遇到错误的情况，请参考博客：http://doc00.com/doc/1001007m9
